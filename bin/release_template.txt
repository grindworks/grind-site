Release title
v1.6.6 - License Registration Fix, Admin UI Improvements & Translation Updates

Release notes
This release primarily fixes an issue where the commercial license key could not be registered successfully. It also brings various improvements to the admin interface, enhances the contact forms across all bundled themes, and updates translation files.

### 🛡️ Bug Fixes
- **License Registration:** Fixed an issue where the license key validation and registration failed under certain conditions.
- **System & Functions:** Minor bug fixes and logic improvements in core functions (`posts.php`, `system.php`, `theme.php`, `utils.php`).

### ✨ Enhancements
- **Admin UI & Editor Improvements:** Updated various block views and the block editor script (`admin_editor.js`) for better stability and user experience.
- **Contact Form Updates:** Enhanced the contact form templates (`contact.php`) across all default themes (Default, Bootstrap, Corporate, Marketing, Neo-Minimalist, Photographer).
- **Translation Updates:** Updated language files (`en.php`, `ja.php`, `de.php`, `es.php`, `fr.php`, `pt-br.php`) with new keys and improved translations.

---

## 🇯🇵 日本語 (Japanese)

このリリースでは、商用ライセンスが正常に登録できない不具合の修正をメインに、管理画面UIの各種改善、全テーマのお問い合わせフォームの強化、および多言語翻訳ファイルのアップデートが含まれています。

### 🛡️ バグ修正
- **ライセンス登録の修正:** 特定の環境下において、商用ライセンスキーが正常に登録・認証されない不具合を修正しました。
- **システムとコア関数:** コア機能（`posts.php`, `system.php`, `theme.php`, `utils.php`）における軽微なバグ修正とロジックの改善を行いました。

### ✨ 新機能・改善
- **管理画面＆エディタの改善:** エディタの安定性向上とUX改善のため、各種ブロックのビューファイルおよびエディタのスクリプト(`admin_editor.js`)を更新しました。
- **お問い合わせフォームのアップデート:** 同梱されている全テーマ（Default, Bootstrap, Corporate, Marketing, Neo-Minimalist, Photographer）のお問い合わせフォームテンプレート (`contact.php`) を改良しました。
- **翻訳ファイルの更新:** 各国語の言語ファイル (`en.php`, `ja.php`, `de.php`, `es.php`, `fr.php`, `pt-br.php`) に新しいキーを追加し、翻訳の精度を向上させました。
