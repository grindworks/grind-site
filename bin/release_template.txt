Release title
v1.6.0 - Post Revisions, Advanced SSG & Custom Fields

Release notes
This major release introduces Post Revisions, advanced Static Site Generation with a Virtual Publish Queue, Custom Fields powered by `theme.json`, and new Affiliate plugins for eBay and Rakuten.

### ✨ New Features & Enhancements
- **Post Revisions:** Never lose your work again. Automatically saves a history of your post edits, allowing you to restore previous versions seamlessly.
- **Custom Fields (`theme.json`):** Developers can now define custom fields directly in their theme's `theme.json` file. Create tailored editing experiences for specific post types or categories without writing PHP.
- **Virtual Publish Queue (Advanced SSG):** Static Site Generation now features a robust background queuing system, significantly improving build reliability and performance for large sites.
- **HTML Block Live Preview:** Added a live preview toggle for the HTML block, making it easier to embed and verify custom code, iframes, and affiliate tags directly within the editor.
- **Markdown Export:** Export your entire site content as clean Markdown files, perfect for AI analysis, RAG pipelines, or migrating to other platforms.
- **New Affiliate Plugins:** Expanded monetization options with new shortcode plugins for eBay (`_ebay_affiliate.php`) and Rakuten (`_rakuten_affiliate.php`).
- **New Admin Skins:** Added two stunning new dark mode skins: "Neon Matrix" and "Obsidian Glass".

### 🛡️ Security & Stability
- **Enhanced Security:** Continued hardening of core systems, including improved sanitization and validation across all new features.
- **Bug Fixes:** Resolved various edge cases and UI quirks based on community feedback.

## 🇯🇵 日本語 (Japanese)

今回のメジャーリリースでは、記事のリビジョン（変更履歴）機能、バックグラウンド処理（キュー）を備えた高度な静的サイト生成（SSG）、`theme.json` で定義可能なカスタムフィールド、そして新しいアフィリエイトプラグイン（eBay、楽天）などの強力な新機能が導入されました。

### ✨ 新機能・改善
- **記事のリビジョン機能:** 編集のたびに履歴が自動的に保存され、過去のバージョンへシームレスに復元できるようになりました。
- **カスタムフィールド (`theme.json`):** テーマ内の `theme.json` ファイルでカスタムフィールドを定義できるようになりました。PHPを書くことなく、投稿タイプやカテゴリに応じた専用の入力項目を作成可能です。
- **仮想パブリッシュキュー (高度なSSG):** 静的サイト生成（SSG）にバックグラウンドキューシステムが導入され、大規模サイトにおけるビルドの安定性とパフォーマンスが大幅に向上しました。
- **HTMLブロックのライブプレビュー:** HTMLブロックにライブプレビュートグルを追加し、カスタムコードやiframe、アフィリエイトタグの表示確認がエディタ上で直接行えるようになりました。
- **Markdown一括エクスポート:** サイトの全コンテンツをクリーンなMarkdownファイルとしてエクスポートする機能を追加。AI分析（RAG等）や他プラットフォームへの移行に最適です。
- **新アフィリエイトプラグイン:** eBay (`_ebay_affiliate.php`) と楽天 (`_rakuten_affiliate.php`) 向けのアフィリエイトショートコードプラグインを新たに追加し、収益化の選択肢を拡大しました。
- **新管理画面スキン:** 美しいダークモードスキン「Neon Matrix」と「Obsidian Glass」を追加しました。

### 🛡️ セキュリティと安定性
- **セキュリティの強化:** 新機能を含むシステム全体のサニタイズとバリデーションをさらに強化しました。
- **バグ修正:** コミュニティのフィードバックに基づき、さまざまなエッジケースやUIの細かな問題を修正しました。
