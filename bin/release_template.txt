Release title
v1.7.0 - Plugin Management UI & Fullscreen Presentation Theme

Release notes
This release introduces a brand new **Plugin Management UI**, allowing administrators to easily enable and disable plugins directly from the dashboard. We've also added a new **Fullscreen Presentation** theme, new admin skins, and completed comprehensive translation synchronizations.

### ✨ New Features & Enhancements
- **Plugin Management UI:** A new dashboard interface to securely toggle system plugins on and off without touching server files.
- **New Theme & Skins:** Added the `fullscreen_presentation` theme and several new retro-arcade admin skins (e.g., `arcade-lode-digger`).
- **Translation Sync:** Complete synchronization of English, Japanese, French, Spanish, Portuguese, and German translation keys across the core and all 10 themes.

### 🛡️ Bug Fixes & Maintenance
- **System Stability:** Resolved translation key discrepancies and verified perfect HTML/SVG rendering syntax across the system.
- **Easter Egg:** Added a new developer-focused Easter egg plugin that outputs a cool system boot sequence in the browser console.

## 🇯🇵 日本語 (Japanese)

このリリースでは、ダッシュボードから直接プラグインを有効・無効化できる **「プラグイン管理UI」** を新たに搭載しました。また、新しい「フルスクリーンプレゼンテーション」テーマや管理画面スキンを追加し、全言語の翻訳同期を完了しました。

### ✨ 新機能・改善
- **プラグイン管理UI:** サーバーのファイルを直接操作することなく、管理画面から安全にプラグインの有効/無効を切り替えられるインターフェースを追加しました。
- **新テーマ・スキン:** 新しいテーマ `fullscreen_presentation` と、レトロアーケード風の管理画面スキン（`arcade-lode-digger` など）を追加しました。
- **翻訳の完全同期:** コアおよび全10テーマにおいて、英語・日本語・フランス語・スペイン語・ポルトガル語・ドイツ語のすべての翻訳キーを完全に同期しました。

### 🛡️ バグ修正・メンテナンス
- **システムの安定性:** 翻訳キーの不足を解消し、HTML構文やSVGレンダリングの細かな問題を検証・修正しました。
- **イースターエッグ:** ブラウザのコンソールにシステム起動シーケンスを出力する、開発者向けの新しいイースターエッグプラグインを追加しました。
