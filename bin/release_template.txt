Release title
v1.6.8 - Contact Form Anti-Spam, SSG Improvements & Password Protect Enhancements

Release notes
This release introduces powerful anti-spam measures for the contact form, improves Static Site Generation (SSG) output, and enhances the Password Protect block.

### ✨ New Features & Enhancements
- **Contact Form Anti-Spam:** Added privacy policy agreement checkbox, honeypot fields, and time-trap mechanisms to significantly reduce spam submissions.
- **SSG & Feeds:** Optimized URL resolution and absolute/relative path conversion during Static Site Generation. Improved output formats for RSS, Sitemap, and AI crawler files (`llms.txt`).
- **Password Protection:** Added bypass functionality for administrators and preview modes to the Password Protect block.
- **Affiliate Plugins:** Refined shortcode processing and UI for Amazon, Rakuten, and eBay affiliate plugins.
- **CLI Tools:** Enhanced the `grind.php` CLI tool, including improvements to the Markdown export and migration package generation processes.

### 🛡️ Bug Fixes & Maintenance
- **Editor UI:** Improved drag & drop and paste behaviors in `admin_editor.js` and `media_manager.js`.
- **CSS/Styles:** Updated Tailwind CSS builds across the admin panel and all built-in themes.
- **Translations:** Added new translation keys for contact form and password protect features across multiple languages.

## 🇯🇵 日本語 (Japanese)

このリリースでは、コンタクトフォームの強力なスパム対策の実装、静的サイト生成（SSG）の最適化、パスワード保護ブロックの利便性向上を行いました。

### ✨ 新機能・改善
- **コンタクトフォームのスパム対策強化:** プライバシーポリシーの同意チェックボックス、ハニーポット、タイムトラップ（送信時間計測）機能を追加し、スパムを強力にブロックします。

- **パスワード保護ブロック:** 管理者ログイン時やプレビューモード時に、パスワード入力を省略（バイパス）できる機能を追加しました。
- **アフィリエイトプラグイン:** Amazon、楽天、eBayプラグインのショートコード展開処理と管理画面UIを改善しました。
- **CLIツール:** `bin/grind.php` のMarkdownエクスポート機能やマイグレーションパッケージ作成処理を最適化しました。

### 🛡️ バグ修正・メンテナンス
- **エディタUI:** `admin_editor.js` や `media_manager.js` におけるドラッグ＆ドロップやペースト時の挙動を修正・改善しました。
- **CSS更新:** 管理画面および各テーマのTailwind CSSビルド結果を最新化しました。
