Release title
v1.6.7 - Core Maintenance, Translation Fixes & Skin Cleanup

Release notes
This release focuses on internal maintenance, cleaning up unused admin skin files, fixing missing translation keys, and improving overall system stability.

### 🛡️ Bug Fixes & Maintenance
- **Translations:** Added missing translation keys for the Password Protect block across all languages (`en`, `ja`, `de`, `es`, `fr`, `pt-br`).
- **Block Views:** Cleaned up the view template for the Password Protect block.
- **Dependencies:** Updated internal build dependencies (`caniuse-lite` database).

### ✨ Enhancements
- **Admin Skins:** Cleaned up 24 unused/deprecated admin skins to streamline the system.
- **Core & API:** General refinements and stability improvements in core systems, API endpoints, and block editor components.

## 🇯🇵 日本語 (Japanese)

このリリースは、内部システムのメンテナンス、未使用スキンファイルの整理、翻訳キーの追加、および全体的な安定性向上を目的としています。

### 🛡️ バグ修正・メンテナンス
- **翻訳ファイルの修正:** パスワード保護ブロックに関する多言語翻訳キー (`err_password_protect_at_bottom`) の抜けを全言語で修正しました。
- **ブロックビュー:** パスワード保護ブロックのテンプレートコードをクリーンアップしました。
- **依存パッケージ:** 内部のビルドツール（ブラウザ対応状況データベース）を最新化しました。

### ✨ 改善・新機能
- **管理画面スキン:** システムをスッキリさせるため、未使用・非推奨のカスタムスキン（24種類）を整理・削除しました。
- **コアシステム・API:** データベース処理、ファイルマネージャー、APIエンドポイント、および各種ブロックコンポーネントの全般的なブラッシュアップを行いました。
