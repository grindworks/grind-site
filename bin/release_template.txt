Release title
v1.6.3 - Security Enhancements, Bug Fixes & UI Improvements

Release notes
This release brings important security enhancements, bug fixes for AI exports, and minor UI improvements across the admin panel.

### ✨ Enhancements
- **UI & UX Improvements:** Updated admin layout components (`topbar`, `sidebar`) and block editor scripts for a smoother experience.
- **Sample Data:** Updated sample data to reflect the latest template structures.

### 🛡️ Security & Stability
- **Security Enhancements:** Strengthened XSS protection across various admin blocks, layouts, and plugins by enforcing strict `ENT_QUOTES | ENT_HTML5` encoding.
- **Bug Fixes:**
  - Fixed an issue where `{{CMS_URL}}` placeholders were not properly resolved in AI markdown exports (`llms-full.txt`) and sponsored links.
  - Corrected a variable naming typo in the category management logic to improve code maintainability.

---

## 🇯🇵 日本語 (Japanese)

このリリースでは、より厳格なXSS保護によるセキュリティ強化、AI向けエクスポート等での表示バグ修正に加え、管理画面全体にわたるUIの改善が行われています。

### ✨ 新機能・改善
- **UI/UXの改善:** 管理画面のレイアウト（トップバー、サイドバー）およびブロックエディタの挙動を最適化し、操作性を向上させました。
- **サンプルデータの更新:** 最新のテンプレート構造に合わせてサンプルデータを更新しました。

### 🛡️ セキュリティと安定性
- **セキュリティ強化:** 各種管理画面ブロック、レイアウト、プラグインの出力において、`ENT_QUOTES | ENT_HTML5` エンコーディングを厳格に適用し、XSS保護をさらに強化しました。
- **バグ修正:**
  - AI向けのMarkdown出力（`llms-full.txt`）やスポンサーリンクにおいて、`{{CMS_URL}}` プレースホルダーが正しくURLに変換（解決）されない表示バグを修正しました。
  - カテゴリ管理ロジックの変数名のタイポを修正し、コードの保守性を向上させました。
