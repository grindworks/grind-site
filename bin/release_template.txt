Release title
v1.6.9 - New Affiliate Block (Amazon/Rakuten/eBay) & Editor UX Improvements

Release notes
This release introduces a brand new, highly requested **Affiliate Block** that makes it easier than ever to embed beautiful Amazon, Rakuten, and eBay product cards directly from the block editor.

### ✨ New Features & Enhancements
- **Dedicated Affiliate Block:** Added a new block specifically designed for affiliate marketing. Switch seamlessly between Amazon, Rakuten, and eBay within the editor.
- **Smart ASIN Extraction:** Simply paste an Amazon product URL into the editor, and the system will automatically extract the ASIN and region (e.g., co.jp, com), drastically improving your workflow.
- **Seamless Plugin Integration:** The new block architecture perfectly integrates with existing affiliate shortcode plugins, ensuring high performance and maintainability.

### 🛡️ Bug Fixes & Maintenance
- **Editor UX:** Optimized the block deletion logic (`admin_editor.js`) so that empty affiliate blocks are removed silently without unnecessary confirmation dialogs.
- **Security:** Enforced strict HTML escaping (`htmlspecialchars`) and double-encoding prevention for all affiliate attributes.

## 🇯🇵 日本語 (Japanese)

このリリースでは、多くのご要望をいただいていた専用の **「アフィリエイトブロック」** を新たに搭載しました。Amazon、楽天、eBayの美しい商品カードを、ブロックエディタからこれまで以上に直感的に作成できます。

### ✨ 新機能・改善
- **アフィリエイト専用ブロック:** ブロックエディタにAmazon、楽天、eBayのアフィリエイトリンクを簡単に挿入できる専用ブロックを追加しました。タブで簡単にプラットフォームを切り替えられます。
- **ASIN自動抽出 (Smart Paste):** Amazonの商品URLをペーストするだけで、URLからASINとリージョン（co.jpなど）を自動的に抽出するスマート入力機能を実装しました。これにより作業効率が劇的に向上します。
- **プラグインとのシームレスな統合:** 新しいブロックは既存のアフィリエイトプラグイン（ショートコード）と連携するDRYな設計となっており、高いパフォーマンスと保守性を実現しています。

### 🛡️ バグ修正・メンテナンス
- **エディタUIの最適化:** 空のアフィリエイトブロックを削除する際、不要な確認ダイアログが表示されないよう削除ロジックを最適化しました。
- **セキュリティ:** 属性出力時の二重エンコード防止機能および、厳格なエスケープ処理を徹底し安全性を強化しました。
